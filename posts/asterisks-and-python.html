    Asterisks and Python

    Posted: 2012-05-07 00:00   |  More posts about asterisk Computer function PhD Python imported

    There are many possible uses of the "*" in Python, to be not confused, this is a brief summary of them.

    Math
    As usual, one asterisk means multiplication, two asterisks mean power with integer index: 

    2*3 = 6
2**3 = 8

    Functions
    In functions declarations "a parameter preceded by * in the parameter list for a function will result in any extra positional parameters being aggregated into a tuple [...] and likewise in Python a parameter preceded by ** will result in any extra keyword parameters being aggregated into a dictionary." (from Wikipedia). In a function call the asterisk before a (list or tuple) parameter expand it as the arguments of that function. You can find some examples here.
    Just to give a little example, consider this: 

    In: def pippo(\*args):
        for i in args:
        print i
In: pippo(1,2,3,4)
Out:
1
2
3
4

In: def pippo(a, b, c):
        print a,b,c
In: pluto = [1,2,3]
In: pippo(*pluto)
Out:
1 2 3
