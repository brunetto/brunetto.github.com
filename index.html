Ssh keys exchange    Posted: 2014-01-31 16:46

A superfast note on how to exchange ssh keys in order to connect without the need to remember a password.

On both the computer (A and B):

ssh-keygen
ssh-copy-id -i you@B
ssh you@B # to check if it works

  • locate cuda | grep nvcc
  • locate cuda | grep include
